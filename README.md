# 👨‍💼 Professional HR Analytics Dashboard

A professional HR Analytics Dashboard built using **Python, Pandas, and Matplotlib** to analyze workforce data and visualize key HR metrics through KPI cards and interactive charts.

---

## 📊 Dashboard Features

- 👨‍💼 Total Employees KPI
- 👨 Male Employees KPI
- 👩 Female Employees KPI
- 💰 Average Salary KPI
- ⭐ Employee Performance KPI
- 📉 Employee Attrition Analysis
- 📈 Employee Growth Trend
- 🏢 Department-wise Employees
- 🥧 Gender Distribution
- 💵 Salary Trend
- 📅 Employee Experience Trend
- 😊 Employee Satisfaction Analysis
- 📊 Performance vs Attrition Comparison
- 🌙 Professional Dark Theme

---

## 🛠 Technologies Used

- Python
- Pandas
- Matplotlib
